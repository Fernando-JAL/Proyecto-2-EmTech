# Proyecto-2-EmTech
Synergy Logistics es una empresa dedicada a la intermediación de servicios de importación y exportación de diferentes productos. Se pondrán en práctica las herramientas de Python para el análisis de datos para generar un análisis que sirva de base para la estructuración de una estrategia operativa.

Para ello, se plantea analizar la viabilidad de 3 opciones de enfoque: rutas de importación y exportación, medio de transporte utilizado y valor total de importaciones y exportaciones.

El repositorio consta de:

PROYECTO-02-ALCANTARA-FERNANDO.pdf - Reporte que contiene el análisis de los enfoques propuestos y la conclusión de cuál es la mejor opción para la empresa.

PROYECTO-02-ALCANTARA-FERNANDO.ipynb - Archivo de jupyter que contiene los códigos desarrollados en python. 
  Explorando el enfoque de rutas de exportación e importación; se muestran las 10 rutas más demandadas y las 10 rutas más valiosas acompañado de un gráfico de redes que muestra la utilidad de cada ruta destacando las más valiosas, tanto para las exportaciones como para las importaciones de forma independiente.
  Explorando el enfoque de medios de transporte; se muestra la utilidad para cada medio de transporte utilizado (marítimo, aéreo, ferroviario y carretero) resumiendo el número de envíos, los países que utilizan dicho medio de transporte y finalmente las ganancias generadas por cada medio de transporte. Se concluye graficando la utilidad anual de cada medio de transporte. Este análisis se realizó para las exportaciones e importaciones de forma independiente.
  Explorando el enfoque de valor total de exportaciones e importaciones; dado un porcentaje del total de ingresos, se muestran los países que generan dichos ingresos. Se incluye además, el porcentaje de destinos alcanzados, porcentaje de envíos realizados, porcentaje de productos, porcentaje de compañías y porcentaje de ganancias que dichos países generan. Este análisis se realizó para las exportaciones e importaciones de forma independiente
